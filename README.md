## 构建工具

Amaze UI 使用 [gulp.js](http://gulpjs.com/) 构建项目。

首先全局安装 Gulp：

```
npm install -g gulp
```

然后安装一下依赖：

```
npm install
```

接下来，执行 gulp：

```
gulp
```
